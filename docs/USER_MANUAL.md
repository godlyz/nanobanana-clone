## 快速开始

欢迎来到 Nano Banana 论坛！本手册将帮助您快速上手论坛的各项功能。

### 论坛主要功能

- 📝 **发帖提问**：遇到问题？发帖向社区求助
- 💬 **回复讨论**：分享您的见解，帮助他人解决问题
- 👍 **投票互动**：为优质内容点赞，为不当内容投反对票
- ⭐ **最佳答案**：标记对您最有帮助的答案
- 🔖 **分类浏览**：按分类查找感兴趣的话题
- 🏷️ **标签筛选**：通过标签精准定位相关讨论
- 🔍 **全文搜索**：快速找到您需要的内容

### 谁可以使用论坛？

- **游客**：可以浏览所有公开帖子和回复
- **注册用户**：可以发帖、回复、投票、标记最佳答案
- **版主/管理员**：拥有更多管理权限（编辑/删除他人内容、置顶/锁定帖子等）

## 账号与登录

### 注册账号

1. 点击页面右上角的 **"登录"** 按钮
2. 选择登录方式：
   - **GitHub 登录**：使用 GitHub 账号一键登录
   - **Google 登录**：使用 Google 账号一键登录
3. 授权后自动创建账号并登录

### 完善个人资料

首次登录后，建议完善您的个人资料：

1. 点击右上角头像，进入 **"个人资料"**
2. 设置以下信息：
   - **显示名称**：在论坛中显示的用户名
   - **头像**：上传个人头像（支持 JPG/PNG，最大 2MB）
   - **个人简介**：简单介绍自己

### 退出登录

点击右上角头像 → **"退出登录"**

## 浏览论坛

### 论坛首页

论坛首页展示所有公开帖子，您可以：

1. **查看置顶帖**：重要公告和优质内容会被置顶显示
2. **浏览最新帖子**：按时间排序查看最新讨论
3. **查看热门帖子**：按回复数和投票数排序
4. **筛选未回复帖子**：查看还没有回复的帖子

### 分类浏览

点击左侧边栏的分类，查看该分类下的所有帖子：

- **技术问答**：编程、开发相关问题
- **产品讨论**：产品功能、使用建议
- **公告通知**：官方公告和更新
- **闲聊交流**：轻松话题，自由讨论

### 标签筛选

1. 点击帖子标题下的 **标签**（如 `#JavaScript`、`#Bug`）
2. 自动筛选出所有带该标签的帖子
3. 在侧边栏的 **"热门标签"** 中也可以快速选择

### 排序选项

通过页面上方的排序下拉菜单选择：

- **最新**：按创建时间降序
- **热门**：按回复数 + 投票数综合排序
- **未回复**：只显示没有回复的帖子
- **最热**：按投票数降序

## 创建帖子

### 发帖步骤

1. 点击页面右上角的 **"发帖"** 按钮
2. 填写帖子信息：

   **a. 选择分类**
   - 从下拉菜单中选择合适的分类
   - 分类决定了帖子的归属和展示位置

   **b. 输入标题**
   - 标题长度：3-200 个字符
   - 建议：简洁明了，概括核心问题
   - 示例：
     - ✅ "React Hook 中如何正确使用 useEffect？"
     - ❌ "求助！！！"（过于简略）

   **c. 编写正文**
   - 正文长度：至少 10 个字符
   - 支持 Markdown 格式（详见 [Markdown 编辑器](#markdown-编辑器)）
   - 建议包含：
     - 问题描述
     - 已尝试的解决方法
     - 期望结果
     - 相关代码或截图

   **d. 添加标签（可选）**
   - 最多 5 个标签
   - 标签帮助其他用户快速找到您的帖子
   - 示例：`React`、`JavaScript`、`Bug`、`性能优化`

3. 点击 **"发布"** 按钮

### 发帖建议

**✅ 好的帖子示例**：

```markdown
## 问题描述
在使用 React Hook 的 useEffect 时，遇到无限循环调用的问题。

## 代码
​```jsx
function MyComponent({ userId }) {
  const [data, setData] = useState(null)

  useEffect(() => {
    fetch(`/api/users/${userId}`)
      .then(res => res.json())
      .then(setData)
  }, [data])  // 这里有问题
}
​```

## 已尝试
- 尝试移除依赖数组，但会导致每次渲染都触发
- 查阅官方文档，但没找到类似案例

## 期望
希望了解正确的依赖数组写法，避免无限循环。
```

**❌ 不好的帖子示例**：

```
求助！代码报错了，怎么办？
```
（缺少具体信息，其他用户无法帮助）

## 回复帖子

### 如何回复

1. 打开帖子详情页
2. 滚动到底部，找到 **"写回复"** 输入框
3. 输入回复内容（至少 10 个字符）
4. 点击 **"发布回复"** 按钮

### 回复他人的回复（嵌套回复）

1. 在想要回复的回复下方，点击 **"回复"** 按钮
2. 输入内容
3. 点击 **"发布"**

嵌套回复会显示为缩进样式，便于追踪对话。

### 回复建议

**✅ 好的回复示例**：

```markdown
这个问题的根本原因是依赖数组中包含了 `data`，导致每次 `data` 更新都会触发新的请求。

**解决方法**：
​```jsx
useEffect(() => {
  fetch(`/api/users/${userId}`)
    .then(res => res.json())
    .then(setData)
}, [userId])  // 只依赖 userId
​```

**说明**：
- `userId` 变化时才重新请求
- `data` 更新不会触发新请求
- 详细原理可参考 [React 官方文档](https://react.dev/reference/react/useEffect)
```

**❌ 不好的回复示例**：

```
我也不知道
```
（无价值回复）

## 投票功能

### 为什么要投票？

投票帮助社区：
- **识别优质内容**：高质量的帖子和回复会获得更多赞成票
- **过滤无效信息**：不当或错误的内容会被投反对票
- **排序参考**：投票数是排序的重要依据

### 如何投票

1. **赞成票（Upvote）**：
   - 点击帖子/回复下方的 **👍** 按钮
   - 表示"这个内容有帮助"或"我同意这个观点"

2. **反对票（Downvote）**：
   - 点击帖子/回复下方的 **👎** 按钮
   - 表示"这个内容无帮助"或"内容有误"

3. **取消投票**：
   - 再次点击相同的投票按钮即可取消

4. **切换投票**：
   - 点击相反的投票按钮（例如已投赞成票，点击反对票按钮会切换为反对票）

### 投票规则

- ✅ 每个用户对每个帖子/回复只能投一票（赞成或反对）
- ✅ 可以随时更改或取消自己的投票
- ❌ 不能给自己的内容投票
- ❌ 未登录用户无法投票

### 投票建议

**何时投赞成票**：
- 回复准确解决了问题
- 内容详细、有参考价值
- 提供了有用的资源或链接
- 观点有建设性

**何时投反对票**：
- 内容明显错误或误导
- 无关回复（灌水、广告）
- 态度不友好或攻击性
- 违反社区规则

## 标记最佳答案

### 什么是最佳答案？

最佳答案是帖子作者从所有回复中选出的**最有帮助**的那一个。

### 如何标记最佳答案

1. 只有**帖子作者**可以标记最佳答案
2. 在想标记的回复下方，点击 **"标记为最佳答案"** 按钮
3. 确认后，该回复会：
   - 显示绿色 **⭐ 最佳答案** 标签
   - 自动排到所有回复的最前面
   - 帮助其他有相同问题的用户快速找到答案

### 取消最佳答案

1. 点击已标记最佳答案回复下方的 **"取消最佳答案"** 按钮
2. 确认后，该回复恢复普通状态

### 最佳答案建议

- 尽量在问题解决后及时标记最佳答案
- 选择**真正解决问题**的回复，而不是单纯赞同你的观点
- 最佳答案可以随时更改

## 编辑与删除

### 编辑帖子

1. 打开自己发布的帖子
2. 点击帖子右上角的 **"编辑"** 按钮
3. 修改标题、正文、标签
4. 点击 **"保存"** 按钮

**注意**：
- 只能编辑自己发布的帖子
- 编辑后会显示"已编辑"标记和编辑时间
- 版主/管理员可以编辑所有帖子

### 编辑回复

1. 在自己的回复下方，点击 **"编辑"** 按钮
2. 修改内容
3. 点击 **"保存"** 按钮

### 删除帖子

1. 打开自己发布的帖子
2. 点击帖子右上角的 **"删除"** 按钮
3. 确认删除

**注意**：
- 删除是**软删除**（数据仍保留在数据库，但用户不可见）
- 删除帖子后，该帖子下的所有回复也会被隐藏
- 删除操作**不可恢复**

### 删除回复

1. 在自己的回复下方，点击 **"删除"** 按钮
2. 确认删除

**注意**：
- 删除回复是软删除
- 嵌套回复（子回复）仍会保留
- 删除操作不可恢复

## Markdown 编辑器

### 什么是 Markdown？

Markdown 是一种轻量级标记语言，可以让您快速格式化文本，无需复杂的 HTML 代码。

### 基本语法

#### 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 2. 文本样式

```markdown
**加粗文本**
*斜体文本*
~~删除线文本~~
`代码文本`
```

效果：
- **加粗文本**
- *斜体文本*
- ~~删除线文本~~
- `代码文本`

#### 3. 列表

**无序列表**：
```markdown
- 项目一
- 项目二
- 项目三
```

**有序列表**：
```markdown
1. 第一步
2. 第二步
3. 第三步
```

#### 4. 链接和图片

**链接**：
```markdown
[链接文本](https://example.com)
```

**图片**：
```markdown
![图片描述](https://example.com/image.jpg)
```

#### 5. 代码块

**行内代码**：
```markdown
使用 `console.log()` 打印信息
```

**代码块**（支持语法高亮）：
````markdown
​```javascript
function hello() {
  console.log("Hello, World!")
}
​```
````

效果：
```javascript
function hello() {
  console.log("Hello, World!")
}
```

#### 6. 引用

```markdown
> 这是一段引用文本
> 可以多行
```

效果：
> 这是一段引用文本
> 可以多行

#### 7. 分隔线

```markdown
---
```

### 图片上传

编辑器支持**直接上传图片**：

1. 点击编辑器工具栏的 **"上传图片"** 按钮（📷 图标）
2. 选择图片文件（支持 JPG/PNG/GIF，最大 5MB）
3. 上传后自动插入 Markdown 图片语法

或者：

1. 直接**拖拽图片**到编辑器中
2. 自动上传并插入

### 实时预览

编辑器右侧会实时预览 Markdown 渲染效果，方便您检查格式。

## 搜索功能

### 全文搜索

1. 点击页面顶部的 **搜索框**
2. 输入关键词（至少 2 个字符）
3. 按 **Enter** 或点击搜索按钮
4. 搜索结果会显示匹配的帖子标题和内容摘要

### 高级搜索

在搜索结果页面，您可以进一步筛选：

1. **按分类筛选**：选择特定分类
2. **按标签筛选**：选择特定标签
3. **排序方式**：最新、最热、最相关

### 搜索建议

- 使用**关键词组合**：例如 "React useEffect 无限循环"
- 使用**引号**精确匹配：例如 `"Error: Cannot read property"`
- 尝试**同义词**：例如 "JS" → "JavaScript"

## 常见问题

### 1. 为什么我无法发帖/回复？

**可能原因**：
- 您未登录 → 请先登录
- 帖子已锁定 → 锁定的帖子无法回复
- 帖子已关闭 → 关闭的帖子无法回复
- 帖子已归档 → 归档的帖子无法编辑或回复

### 2. 为什么我的帖子不显示？

**可能原因**：
- 内容正在审核（新用户首次发帖可能需要审核）
- 内容违反社区规则，被版主/管理员隐藏
- 网络问题导致发布失败 → 刷新页面重试

### 3. 如何@ 其他用户？

目前论坛**暂不支持** @ 提及功能，后续版本会添加。

### 4. 如何删除账号？

请联系管理员处理账号删除请求。

### 5. 如何举报不当内容？

1. 点击帖子/回复下方的 **"举报"** 按钮
2. 选择举报原因：
   - 垃圾广告
   - 不友好内容
   - 违反规则
   - 其他
3. 提交后，版主会尽快处理

### 6. 如何订阅帖子更新？

目前论坛**暂不支持**订阅功能，后续版本会添加邮件/通知订阅。

### 7. 如何查看我的历史帖子和回复？

1. 点击右上角头像
2. 选择 **"我的帖子"** 或 **"我的回复"**
3. 查看历史记录

### 8. 为什么我的回复没有被标记为最佳答案？

只有**帖子作者**可以标记最佳答案。如果您的回复确实解决了问题，可以礼貌地提醒帖子作者。

### 9. 如何修改用户名/头像？

1. 点击右上角头像
2. 进入 **"个人资料"**
3. 修改显示名称和头像

### 10. 论坛支持哪些语言？

目前支持：
- 🇨🇳 简体中文
- 🇺🇸 English

点击右上角的语言切换器即可切换。

## 社区规则

为了维护友好、专业的社区氛围，请遵守以下规则：

✅ **鼓励行为**：
- 友善、尊重他人
- 分享有价值的知识
- 提供建设性的反馈
- 帮助新用户

❌ **禁止行为**：
- 发布垃圾广告
- 人身攻击、侮辱
- 发布违法、色情、暴力内容
- 恶意刷屏、灌水
- 冒充他人

违反规则可能导致内容被删除、账号被封禁。
