## Moderation Principles

### Fairness and Impartiality

  - Apply same standards to all users regardless of influence, follower count, or contributions
  - Do not let personal preferences affect moderation decisions

  - Make decisions based on objective evidence only
  - Presumption of innocence - when uncertain, favor content retention

### Consistency

  - Refer to handling of similar past cases
  - Maintain stable moderation standards

  - Consult other moderators on complex or controversial cases
  - Regular team meetings to unify standards

### Transparency

  - When removing content or banning accounts, provide clear reasons
  - Cite specific violated community guideline clauses

  - Document reasons for each decision in moderation system
  - Facilitate appeal handling and quality review

### Respect

  - Maintain professionalism and politeness even when handling violations
  - Do not use insulting language

  - Do not publicly discuss specific user moderation cases
  - Share sensitive information only within internal systems

### Proportionality

  - Penalties should match violation severity
  - First minor violation: warning > temporary restriction > ban

### Detailed Steps

#### Receive Report

- Access `/admin/moderation/queue`
- View pending reports list

- Report target (artwork, comment, user)
- Report type (spam, inappropriate content, copyright infringement, etc.)
- Detailed explanation provided by reporter
- Report time and priority

#### Assess Content

- Read full content (artwork description, comment thread, user profile)
- Check user history and prior violations
- Review related comments and interactions

- Refer to "Violation Identification" section in this manual
- Determine violation level (Level 1-5)

#### Make Decision

- Junior moderators (Level 2+) can independently handle obvious violations

- Complex or serious cases require agreement from at least 2 moderators
- Controversial cases escalated to Community Manager (Level 4)

#### Execute Penalty

- **Remove Content**: Hide from public display
- **Warning**: Send system notification, record violation
- **Restrict Features**: Disable posting, commenting, liking, etc. (1-30 days)
- **Ban Account**: Temporary (7-30 days) or permanent ban

- Select appropriate penalty in moderation dashboard
- Fill in penalty reason (required, visible to user)
- Set penalty duration (if applicable)

#### Notify User

- System automatically sends notification to user notification center
- Notification includes: violating content, violated terms, penalty, appeal link

- Level 3+ violations trigger email notification

#### Document Decision

- System automatically records:
  - Moderator ID and name
  - Processing time
  - Violation type and level
  - Penalty measure
  - Penalty reason

- Add additional background information (visible only to moderation team)
- Mark special cases or cases requiring follow-up

### Special Handling Situations

- Stricter content moderation standards
- Adult content prohibited (even if artistic)
- Notify guardians when violated (if linked)

- First minor violation receives educational warning, no violation record
- Serious violations still handled according to standards

- **No lowered standards** - same rules apply

- Do not adjudicate copyright ownership (not moderator responsibility)
- Only handle obvious infringement (e.g., stealing others' works)

## Appeal Handling

### Appeal Review Process

- User submits appeal via `/appeal` page
- System automatically creates appeal ticket

- Check if appeal is complete (includes appeal reasons, evidence)
- Review original moderation decision and evidence

- Assign to moderator other than original reviewer
- Re-examine case, consider appellant's explanation
- Consult team (complex cases)

- **Uphold**: Original decision correct, appeal rejected
- **Overturn**: Original decision incorrect, restore content/account
- **Reduce**: Original decision too harsh, reduce penalty (e.g., 30 days to 7 days)

- Send detailed result notification to appellant
- If overturned, immediately restore content/account features

### Appeal Success Rate Target

- **Target**: 20-30% of appeals partially or fully supported
- If success rate < 10%, may indicate overly strict moderation standards
- If success rate > 50%, may indicate low initial review quality

## Self-Care

### Mental Health


- Take 10-15 minute breaks every hour of moderation
- Avoid continuous moderation for more than 4 hours

- If you see disturbing content, you can:
  - Talk to supervisor or other moderators
  - Use company-provided counseling services
  - Request break or switch moderation categories

- Outside work hours, don't handle urgent appeals (unless on duty)
- Don't bring work stress home
