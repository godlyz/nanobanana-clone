## 欢迎语

🎨 **欢迎来到Nano Banana社区！**

Nano Banana是一个以AI创作为核心的创意社区，我们致力于为全球创作者提供一个友好、包容、充满灵感的平台。无论你是AI艺术的初学者还是经验丰富的创作者，这里都是你展示才华、交流学习、激发灵感的理想之地。




## 核心价值观

我们的社区建立在以下核心价值观之上：

### 1. 尊重
- 尊重每一位社区成员，无论其背景、经验或观点
- 尊重他人的创作成果和知识产权

### 2. 包容
- 欢迎来自不同文化、语言、技能水平的创作者
- 反对任何形式的歧视和偏见

### 3. 创新
- 鼓励探索AI创作的无限可能
- 支持原创内容和实验性作品

### 4. 互助
- 分享经验和技巧，帮助他人成长
- 共同建设积极向上的创作氛围

### 5. 责任
- 对自己的言行和创作负责
- 维护社区的健康和安全


## 行为准则

### ✅ 鼓励的行为

1. **友善交流** / Friendly Communication
   - 使用礼貌、尊重的语言
   - 建设性地提供反馈和建议

2. **积极互动** / Active Engagement
   - 参与讨论，分享见解
   - 为他人的作品点赞和评论

3. **知识分享** / Knowledge Sharing
   - 分享创作经验和技巧
   - 帮助新手解决问题

4. **版权保护** / Copyright Protection
   - 标注作品来源和使用的AI工具
   - 尊重他人的原创作品

### ❌ 禁止的行为

1. **骚扰和欺凌** / Harassment and Bullying
   - ❌ 人身攻击、辱骂、威胁
   - ❌ 跟踪、骚扰其他用户

2. **仇恨言论** / Hate Speech
   - ❌ 基于种族、性别、宗教、国籍等的歧视性言论
   - ❌ 煽动仇恨或暴力的内容

3. **垃圾信息** / Spam
   - ❌ 重复发布相同或相似内容
   - ❌ 未经许可的广告和推广

4. **欺诈和冒充** / Fraud and Impersonation
   - ❌ 冒充其他用户、名人或机构
   - ❌ 发布虚假信息或误导性内容

5. **滥用功能** / Feature Abuse
   - ❌ 操纵点赞、评论、关注等互动功能
   - ❌ 使用自动化工具进行恶意操作


## 内容规范

### 允许的内容

✅ **AI生成的艺术作品**
- 图像、视频、动画等AI创作内容

✅ **教程和技巧分享**
- 创作经验、工具使用指南、技巧分享

✅ **讨论和交流**
- AI艺术相关的讨论、灵感分享

✅ **作品展示**
- 个人创作的AI艺术作品集

### 禁止的内容

❌ **非法内容** / Illegal Content
- 违反当地法律法规的内容
- 侵犯知识产权的内容（未经授权的盗版、抄袭）

❌ **成人内容** / Adult Content
- 色情、裸露、性暗示的内容
- 针对未成年人的不当内容

❌ **暴力和血腥** / Violence and Gore
- 极端暴力、血腥、虐待的内容
- 自残或伤害他人的内容

❌ **虚假信息** / Misinformation
- 故意传播虚假、误导性的信息
- 恶意篡改AI生成内容以误导他人

❌ **危险内容** / Dangerous Content
- 教唆犯罪、暴力或危险行为的内容
- 推广危险物品或活动的内容


## 创作规范

### AI工具使用规范

1. **透明度 / Transparency**
   - 建议在作品描述中标注使用的AI工具（如Google Veo, DALL-E等）
   - 对于混合创作（AI+人工编辑），说明各部分的创作方式

2. **原创性 / Originality**
   - 鼓励创作独特的prompts和创意
   - 避免简单复制他人的prompt或作品

3. **质量标准 / Quality Standards**
   - 上传清晰、完整的作品
   - 避免上传低质量、未完成的草稿（除非标注为"实验性"）

### 版权和归属

1. **尊重版权 / Respect Copyright**
   - 不上传未经授权的受版权保护的内容
   - 使用参考图片时，确保拥有使用权或符合合理使用原则

2. **标注来源 / Credit Sources**
   - 如果使用了他人的prompt、风格或灵感，请在作品描述中致谢
   - 使用知名艺术家风格时，注明"风格参考：[艺术家名]"

3. **商业用途声明 / Commercial Use Declaration**
   - 如果作品用于商业目的，请在隐私设置或描述中说明
   - 遵守Nano Banana的服务条款关于商业用途的规定


## 社交互动规范

### 评论规范

✅ **建设性评论** / Constructive Comments
- 提供具体、有帮助的反馈
- 指出作品的亮点和可改进之处

✅ **礼貌用语** / Polite Language
- 使用友好、尊重的语气
- 即使不喜欢作品，也要保持礼貌

❌ **禁止** / Prohibited
- 恶意批评、人身攻击
- 垃圾评论（如"顶""沙发"等无意义内容）

### 关注和互动

✅ **真诚互动** / Genuine Engagement
- 关注真正感兴趣的创作者
- 点赞和评论感兴趣的作品

❌ **禁止** / Prohibited
- 互粉交易（"关注我，我就关注你"）
- 使用自动化工具批量关注/点赞

### 私信规范

✅ **尊重边界** / Respect Boundaries
- 发送友好、专业的消息
- 尊重他人的回复时间

❌ **禁止** / Prohibited
- 骚扰、威胁性消息
- 未经许可的商业推广


## 举报和审核

### 如何举报违规内容

如果你发现违反社区规范的内容，请通过以下方式举报：

**1. 作品举报** / Report Artwork
- 点击作品详情页的"举报"按钮
- 选择违规类型（垃圾信息、不当内容、版权侵犯等）
- 提供详细说明（可选）

**2. 评论举报** / Report Comment
- 点击评论旁的"举报"图标
- 选择违规原因

**3. 用户举报** / Report User
- 访问用户主页，点击"举报用户"
- 描述违规行为和证据

**4. 紧急情况** / Emergency
- 对于严重违规（如非法内容、暴力威胁），请立即发送邮件至：
  - 主题标注：【紧急举报】/ Subject: [Urgent Report]

### 审核流程

1. **自动过滤** / Automated Filtering
   - 系统自动检测和隐藏明显违规内容

2. **人工审核** / Manual Review
   - 审核团队在24小时内审查所有举报
   - 复杂案例可能需要更长时间

3. **处理结果** / Action Taken
   - 举报者将收到处理结果通知
   - 违规者将收到警告或处罚通知

### 申诉机制

如果你认为自己的内容被错误删除或账号被误封，可以：

1. 访问 `/appeal` 页面提交申诉
2. 提供详细的申诉理由和证据
3. 申诉团队将在7个工作日内审查并回复




## 违规处理

### 警告系统

我们采用三级警告制度：

**第一次违规** / First Violation
- ⚠️ 警告 + 移除违规内容
- 保留账号功能

**第二次违规** / Second Violation
- ⚠️⚠️ 警告 + 7天功能限制（禁止发帖、评论）
- 违规记录保留30天

**第三次违规** / Third Violation
- 🚫 永久封禁账号
- 所有内容移除

### 严重违规立即封禁

以下行为将导致立即封禁，不经警告：

- 儿童色情、人口贩卖等严重非法内容
- 真实威胁或暴力煽动
- 大规模垃圾信息攻击
- 恶意使用自动化工具破坏社区
