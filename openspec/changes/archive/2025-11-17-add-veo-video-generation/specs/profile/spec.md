## ADDED Requirements

### Requirement: Video Generation History Display
Personal center SHALL display user video generation history with unified image and video record list.

#### Scenario: View Video Generation History
- **WHEN** user accesses "Generation Records" tab in personal center
- **THEN** system displays unified list with content type filtering (All/Images/Videos), shows thumbnail, prompt, parameters, status, credit cost, timestamp for each video record, and provides play button for fullscreen playback
